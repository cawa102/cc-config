# Init Docs

プロジェクトのdocsディレクトリ構造を初期化する。

## ディレクトリ構成

以下のディレクトリを作成する（既存ならスキップ）:

- `docs/plans/` — 実装計画書
- `docs/search/` — 調査・リサーチ結果
- `docs/discussions/` — ディスカッション記録
- `docs/codex/` — Codexからの評価レポート

## 手順

1. `mkdir -p` で4つのディレクトリを作成
2. 作成結果を報告:

```
docs/
├── plans/        — 実装計画書
├── search/       — 調査・リサーチ結果
├── discussions/  — ディスカッション記録
└── codex/        — Codex評価レポート
```

3. 各ディレクトリに既存ファイルがあれば、その数も表示する
